# Quick start

  1. Download CodeNotary vcn. There are releases for different platforms:
  • Download the latest release and then read the Usage section below.
  • We recommend storing vcn in your PATH - Linux example:
    cp vcn-v0.6.1-linux-amd64 /usr/local/bin/vcn

  1. Authenticate digital objects You can use the command as a starting point.

    vcn authenticate <file|ds://directory|docker://dockerimage|git://gitdirectory>

  2. Create your identity (free) You need an identity at CodeNotary to notarize objects yourself (btw. we're a data-minimum company and only ask for data that is really required)

  3. Notarize existing digital objects Once you have an account you can start notarizing digital assets to give them an identity.

    vcn login
vcn notarize <file|ds://directory|docker://dockerimage|git://gitdirectory>
